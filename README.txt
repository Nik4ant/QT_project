Это приложение по типу Trello, но только с ограниченным функционалом.
Проект разбит на модули, вот их описание:

- data_modules (отвечает за работу с разными данными):
    - db.py: Работает с БД. Выполняет запросы по получению или
    изменению данных БД
    - json.py: Работает с json. В большинстве случаев используется как
    обвёртка над ujson, чтобы не работать с json явно каждый раз. Позволяет
    сделать код чище и удобнее для чтения

- static: В этой папке хранятся статические данные, которые используются
другими файлами. Например файлы интерфейса сгенерированные pyuic, иконки и звук
воспроизводящийся при переносе карточки

- ui (содержит файлы, представляющие собой взаимодействие с интерфейсом и его вывод):
    - login_form.py: Отвечает за форму входа в аккаунт. Можно создать аккаунт, войти в существующий, либо
    продолжить с данными на устройстве.
    - board_selection: Отвечает за окно с выбором доски. По факту просто промежуточная часть между
    формой логина и редактированием сомой доски
    - board_edit: Главный элемент отвечает за редактирование выбранной доски.
    Внутри него буду осуществляться все действия по редактированию доски.

- main.py: Файл с которого начинается работа всей программы, добавляется папка в которой python
может брать файлы и импортировать модули. Так же файл синхронизирует переход между формой логина
и меню с выбором доски для редактирования

- boards.json: Файл с информацией о досках

- last_board.json: Файл с данными о доске, которую редактировали последней.

- user.sqlite: Файл БД со всеми пользователями

внутри файлов модулей есть документация с кучей иформации, которая подсвечивается IntelliSense)

Вещи, которые были измены/добавлены с момента выступления:
    - Теперь полностью одинаковые доски будут удалятся (как дубликаты, чтобы не занимать место)
    - Теперь при выводе списка досок, программа берёт доски отдельно с аккаунта и устройства, чтобы это не выглядило странно
    (потому, что на практике это удобно, но со стороны выглядит странно)
    - Теперь доски с одинаковым названием, но разным содержимым не будут удалены
    - Теперь в меню с выбором доски увеличина область клика по названию доскии (т.е теперь проще выбрать доску)
    - В меню с редактированием доски отображается название редактируемой доски (в navbar'е)
    - Авто сохранение при закрытии окна (но с учётом того, куда зайдёт пользователь)
    - Улучшена адаптивность дизайна
    - Цвет выделения текста внутри описания карточки заменён на светло-серый (для красоты)
    - Появилась возможность указывать имя доски при её создании
    - Лимит по группам был удалён
    - Оптимизация в некоторых частях кода
    - Исправлены ошибки в разных частях программы
    - Внутри кода были поправленны комментарии и у некоторых методов изменилась документация
